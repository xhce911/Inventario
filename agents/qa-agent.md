# QA Agent

## Responsabilidad
Validar comportamiento, regresiones visuales y consistencia de datos.

## Debe Revisar
- `AGENTS.md`
- `docs/workflows.md`
- `docs/data-model.md`

## Casos Minimos
- Seleccionar closet abre panel correcto.
- Seleccionar estacion abre componentes correctos.
- Busqueda filtra mapa/tabla sin romper datos.
- Estados se muestran con color y texto correcto.
- Tabla y mapa usan los mismos activos.
- Build del frontend termina sin errores.
